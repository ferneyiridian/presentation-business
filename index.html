Efficiency Analysis

Walking Waste

Associates spend 18% of their time walking between locations due to inefficient store layout and poor SKU zoning.

Assistance Time

22.5% of time is spent on non-value activities like staging items and seeking assistance for substitutions.

Average Time per Order: 5.3 min

Time Breakdown

Hand-off (66%) 3.5 min
Walking (19%) 1.0 min
Loading (NVA) (15%) 0.8 min

Key Observations

Hand-off process is efficient at 3.5 minutes per order

Significant time wasted in manual loading of orders

Walking time could be reduced with better staging area design

Eliminating loading time could save 15% of dispensing time

Opportunity:

Pre-staging orders in accessible racks could eliminate manual loading time.

Root Cause Analysis – 5 Whys

Problem Statement: Delayed picking due to inefficient routing

Why are routes inefficient?

Items are scattered throughout the store with no optimized path

Why are items scattered?

No clustering logic in the current system

Why no clustering logic?

System doesn't use order data to optimize pickwalks

Why no optimization tools?

No forecasting or optimization tools installed in current system

Solution Implementation

Deploy route optimization software with clustering algorithms

Root Cause Identified

The fundamental issue is the lack of advanced optimization tools in the current system, leading to inefficient routing and increased walk times.

Process Redesign Recommendations

Routing Optimization

  • Deploy clustering logic for efficient pick paths
  • Implement route optimization algorithms

Technology Upgrade

  • Install stronger access points for Telxon reliability
  • Use barcode scans for real-time inventory accuracy

Layout Reorganization

  • Place high-frequency SKUs near Omni zone
  • Create dedicated staging areas by temperature zone

Staffing Optimization

  • Predictive scheduling based on order volume
  • Cross-training for peak time flexibility

Process Improvement

  • Pre-stage totes using racks to eliminate manual loading
  • Implement substitution approval workflow
  • Create standardized work instructions
